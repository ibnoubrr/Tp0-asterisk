Installation Asterisk 20 LTS

Déploiement complet sur Ubuntu 20.04 LTS

Auteur: Ibnou
Version: 20.x (LTS)
Date: 02 Juillet 2022
01

Mise à jour et Prérequis

On prépare le système en installant les outils nécessaires à la compilation des sources.

ibnou@ubuntu:~$ sudo apt update && sudo apt upgrade -y ibnou@ubuntu:~$ sudo apt install wget build-essential subversion -y
02

Téléchargement d'Asterisk 20

Nous téléchargeons la dernière version stable Long Term Support (LTS).

ibnou@ubuntu:~$ cd /opt/ ibnou@ubuntu:/opt$ sudo wget http://downloads.asterisk.org/pub/telephony/asterisk/asterisk-20-current.tar.gz ibnou@ubuntu:/opt$ sudo tar zxf asterisk-20-current.tar.gz ibnou@ubuntu:/opt$ cd asterisk-20.*/
03

Scripts et Dépendances

Exécution des scripts officiels pour installer les bibliothèques manquantes (OpenSSL, LibXML2, etc.).

# Téléchargement des sources MP3 ibnou@ubuntu:/opt/asterisk-20# ./contrib/scripts/get_mp3_source.sh # Installation automatique des dépendances système ibnou@ubuntu:/opt/asterisk-20# ./contrib/scripts/install_prereq install
04

Compilation

ibnou@ubuntu:/opt/asterisk-20# ./configure ibnou@ubuntu:/opt/asterisk-20# make menuselect (Sélectionnez format_mp3) ibnou@ubuntu:/opt/asterisk-20# make -j2 ibnou@ubuntu:/opt/asterisk-20# sudo make install ibnou@ubuntu:/opt/asterisk-20# sudo make basic-pbx ibnou@ubuntu:/opt/asterisk-20# sudo make config
05

Sécurité de l'Exécution

Création d'un utilisateur système sans privilèges pour faire tourner le service Asterisk.

ibnou@ubuntu:~$ sudo adduser --system --group --home /var/lib/asterisk --no-create-home asterisk ibnou@ubuntu:~$ sudo chown -R asterisk: /var/{lib,log,run,spool}/asterisk /usr/lib/asterisk /etc/asterisk
06

Lancement du Service

ibnou@ubuntu:~$ sudo systemctl start asterisk ibnou@ubuntu:~$ sudo systemctl enable asterisk # Accès à la console de debug ibnou@ubuntu:~$ sudo asterisk -rvvvvvvv
Félicitations ! Votre serveur Asterisk 20 est maintenant opérationnel. N'oubliez pas d'ouvrir le port 5060 (UDP) dans votre pare-feu pour autoriser le trafic SIP.
